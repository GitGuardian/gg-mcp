# IDE Rules Directory

This directory holds various rules implemented for each IDE to guide them to perform a remediation cycle, from detection with ggshield to remediation with various tools our MCP serve is offering and with contextual data from the company GitGuardian workspace

## Overview

The rules in this directory enable IDEs to:
- Detect security issues using ggshield and.or MCP scan tool
- Access contextual remediation data from your GitGuardian workspace via MCP tools
- Guide developers through the remediation process
